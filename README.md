# visualize and update an employee organization chart interactively with filtering and drag & drop functionality.



Steps to Run

1. npm i create-react-app
2. npm install
3. npm start


Tech stacks used:

  Html and Css,
  Bootstrap, 
  Tippy.js, 
  React, 
  Go.js.
